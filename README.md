## ğŸ“¦ Instalasi Cepat

1. **Clone & Install**
   ```bash
   git clone <repository-url>
   cd pandulungan
   composer install
   npm install
   ```
2. **Setup Environment**
   ```bash
   cp .env.example .env
   php artisan key:generate
   # Edit .env untuk database dan Fonnte API
   ```
3. **Migrate & Seed**
   ```bash
   php artisan migrate --seed
   ```
4. **Run the App**
   ```bash
   npm run dev
   php artisan serve
   php artisan queue:work
   ```
